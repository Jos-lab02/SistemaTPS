# Sistema de Agendamiento de Citas Médicas (SATM)
## Descripción
Este proyecto es una aplicación desarrollada en Java que permite gestionar citas médicas entre doctores y pacientes mediante una interfaz gráfica (Java Swing).
El sistema permite registrar usuarios, asignar consultorios, configurar horarios y agendar citas, almacenando la información en archivos de texto.
## Funcionalidades principales
El sistema permite:
- Registro e inicio de sesión de usuarios (doctor y paciente)
- Gestión de perfiles
- Asociación y eliminación de consultorios
- Configuración de horarios por consultorio
- Agendamiento de citas médicas
- Visualización de citas por parte del doctor
- Cancelación de citas
- Consulta de citas por parte del paciente
## Requisitos
Para ejecutar el proyecto se necesita:
- Java JDK 8 o superior
- IDE (IntelliJ IDEA, Eclipse o similar)
- Sistema operativo compatible con Java
## Ejecución
1. Descargar el .zip
2. Extarer el contenido
3. Abrir el archivo descomprimido en el IDE
